1. Приказати имена свих девојчица у одељењу III1.

.. code-block:: sql

   SELECT ime
   FROM ucenik
   WHERE pol = 'ж' AND razred = 3 AND odeljenje = 1;

Извршавањем упита добија се следећи резултат:

.. csv-table::
   :header:  "ime"
   :align: left

   Николина
   Вида
   Анђела
   Драгица
   Тања
   ...
